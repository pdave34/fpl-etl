# FPL ETL

A lightweight ETL pipeline that extracts data from external APIs (e.g., Fantasy Premier League), cleans it, and passes PyArrow tables for downstream analytics.

## Features

- API extraction via `APIExtractor` and subclasses.
- Oracle DDL generation.
- Configurable pipeline orchestration.

## Installation

```bash
pip install -e .
```

## Documentation

Run `mkdocs serve` to view the documentation locally.
